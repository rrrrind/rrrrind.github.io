富士通株式会社
Hack the Future：「with コロナ／after コロナ」における社会課題の解決について考えよう！
LINK | GitHub | SLIDE | DEMO
概要：
外食を控え自炊を促すために，冷蔵庫内の食材画像から深層学習によってその日の献立を提案するLINE Botを作成した．具体的には，LINEで『今日の献立は』とメッセージを送信すると，冷蔵庫内のカメラ(ラズパイ)が食材の写真を撮り，その写真が深層学習モデルに入力されることで，冷蔵庫内の食材から作れそうなレシピを予測し，楽天レシピAPIを用いることでレシピのURLをユーザーに返すBotである．対象はレシピの幅が狭い1人暮らしの方や，日々の食事を考えるのが大変な主婦層，家族持ちとなっている．詳細はスライドを参照．最優秀賞受賞

開発人数： 4人＋2人(メンター)
開発期間： 8日間
担当役割： アイデア / データセットの作成 / 深層学習モデルの構築(fine-tuning) / 献立推薦APIの作成 / スライドの作成
使用技術： Docker, Python (Tensorflow2, Flask)
